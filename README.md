# Photography Business Website

## Overview
This project is a multi-page website I built as a final project for a web development course. It is a mock business website created for a photographer, designed to represent how a small creative business could present its services, portfolio, and contact information online.

## What It Does
- Provides a homepage for the business
- Includes an about page to introduce the photographer and brand
- Includes a portfolio page to showcase work
- Includes a contact page for client inquiries
- Uses separate CSS and media folders to keep the project organized

## Project Structure
- `index.html` - homepage
- `about.html` - business and photographer background page
- `portfolio.html` - featured work and gallery page
- `contact.html` - contact page
- `css/` - styling files
- `media/` - images and other visual assets

## Skills Demonstrated
- HTML
- CSS
- Multi-page website structure
- Navigation and page linking
- Layout design
- Front-end project organization

## Why I Included This
I included this project because it shows my ability to build a complete front-end website for a real-world style use case. Even though it was created as a class final project, it was designed around a business concept for someone else rather than just being a generic assignment.
